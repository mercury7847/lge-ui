tooltip 정의

주의사항
  1. gulp-file-include 방식으로 작업 시, (@@)include -> @@include로 적용해야함. xmp 내부에 include 되지 않도록 ()처리함.

1. tooltip (컨텐츠가 한줄짜리 텍스트인 경우)

자세히 보기

1년간 500만원 이상 구매 시 '우수' 고객으로 상향됩니다.
<!-- DOM --> <div class="tooltip-wrap"><!-- 활성화시 active --> <span class="tooltip-icon ui_tooltip-target">자세히 보기</span> <span class="tooltip-box"> <p>1년간 500만원 이상 구매 시 '우수' 고객으로 상향됩니다.</p> <button type="button" class="btn-close"><span class="blind">닫기</span></button> </span> </div> <!-- // DOM --> <!-- page 내 작업 방식 --> <!-- (@@)include ()삭제 후 사용 --> (@@)include('./common/modules/tooltip.html', { "type" : "text", <!-- 텍스트만 한줄 들어갈 경우 --> "text" : "1년간 500만원 이상 구매 시 '우수' 고객으로 상향됩니다." <!-- p태그 안에 들어갈 텍스트만 입력 --> }) <!-- // page 내 작업 방식 -->
