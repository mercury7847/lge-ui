버튼 정의

주의사항
  1. button size PC일 때 194px, 모바일일 때 100% default
  2. 버튼 2개이상 나열될 경우 .btn-group안에 나열, 버튼 2개가 default
  3. 버튼그룹 3개 이상 케이스 발생할 경우 추가 class 생성예정
  4. hover, active 디자인 작업 후 반영 예정

1. button default

문의하기

이전
<!-- default --> <button class="btn"><span>문의하기</span></button> <a href="#n" class="btn"><span>문의하기</span></a> <!-- // default--> <!-- gray button --> <button class="btn gray"><span>이전</span></button> <a href="#n" class="btn gray"><span>이전</span></a> <!-- // gray button -->

1-2. button group

<!-- btn-group (default 2) --> <div class="btn-group"> <button class="btn gray"><span>이전</span></button> <button class="btn"><span>문의하기</span></button> </div> <!-- // btn-group (default 2) -->

2. text link button

로그인하기
<!-- text button --> <a href="#n" class="txt-btn">로그인하기</a> <!-- // text button -->
