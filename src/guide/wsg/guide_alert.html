1. Alert Case - 타이틀

<!-- DOM --> <button type="button" data-id="#laypop1" data-control="modal" class="btn"><span>Alert- 타이틀</span></button> <script> $(function() { $('button').on('click', function(e){ var id = $(e.currentTarget).data('id'); var obj ={title:'.', ok : function (){ }}; var desc = ''; if(id=="#laypop1"){ obj = $.extend(obj,{title:'인증이 완료되지 않았습니다.<br>다시 시도해주세요.'}); desc = ''; } lgkorUI.alert(desc, obj); }); }); </script>

2. Alert Case - 타이틀 + 설명
