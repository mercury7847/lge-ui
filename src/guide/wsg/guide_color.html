사용 컬러 정의

주의사항
  1. 작업중

1. Basic Color

베이직 색상은 모바일의 기본 요소인 텍스트, 라인, 배경 등에
사용되는 색상으로 우측에 제시된 색상 코드 값을 사용한다.

2. Primary Color

Primary Color는 LG전자의 강조되는 콘텐츠를
포인트 컬러로 표현하기 위한 색상

3. Sub Color

Secondary Color는 Primary Color를 뒷받침하는 서브 컬러
