bullet list 정의

주의사항
  1. gulp-file-include 방식으로 작업 시, (@@)include -> @@include로 적용해야함. xmp 내부에 include 되지 않도록 ()처리함.
  2. .bullet-list > .b-tit(타이틀), .b-txt(리스트)
  3. 'bulletTit' context 유무 체크해서 DOM구조 다르게 build됨.
  4. bullet list상단 border있을 경우 "topBorder" : true

1. bullet list(title+list)

무상 서비스 적용기준
무상 서비스 기간 : 설치 일자 기준 3년 적용
VIP 고객 산정 : 멤버십 가입한 고객의 1년간 구매금액 합산 관리
고장성 불량에 한해서 적용(소비자 과실에 의한 서비스는 미 적용), 온라인 몰 구매제품 제외
<!-- DOM --> <dl class="bullet-list"> <dt class="b-tit">무상 서비스 적용기준</dt> <dd class="b-txt">무상 서비스 기간 : 설치 일자 기준 3년 적용</dd> <dd class="b-txt">VIP 고객 산정 : 멤버십 가입한 고객의 1년간 구매금액 합산 관리</dd> <dd class="b-txt">고장성 불량에 한해서 적용(소비자 과실에 의한 서비스는 미 적용), 온라인 몰 구매제품 제외</dd> </dl> <!-- // DOM --> <!-- page 내 작업 방식 --> <!-- (@@)include ()삭제 후 사용 --> (@@)include('./common/modules/bullet-list.html', { "bulletTit" : "무상 서비스 적용기준", <!-- 블릿 타이틀이 있을경우 true, 아닐경우 삭제 --> "bulletTxt" : [ "무상 서비스 기간 : 설치 일자 기준 3년 적용", "VIP 고객 산정 : 멤버십 가입한 고객의 1년간 구매금액 합산 관리", "고장성 불량에 한해서 적용(소비자 과실에 의한 서비스는 미 적용), 온라인 몰 구매제품 제외" ] }) <!-- // page 내 작업 방식 -->
