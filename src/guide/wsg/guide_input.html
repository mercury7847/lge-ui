input 정의

주의사항
  1. 가이드 페이지에서는 기본적으로 display:inline-block, vertical-align:top, width:auto값이 기본이며, 컨텐츠 및 디자인에 따라 필요 시 input-wrap에서 재정의 해주시면 됩니다.
  2. count, file upload 등 input과 다른 요소가 함께 들어가는 경우 반응형 대응을 위해 .input-wrap > .box로 감싸 display:table로 작업 진행되었습니다.

1. Default input, textarea






<!-- default --> <span class="input-wrap"> <input type="text" placeholder="이름을 입력하세요."> </span> <span class="input-wrap"> <textarea placeholder="내용을 입력해 주세요."></textarea> </span> <!-- // default --> <!-- readonly, disabled --> <span class="input-wrap"> <input type="text" value="읽기 전용" readonly> </span> <span class="input-wrap"> <input type="text" value="입력 불가" disabled> </span> <!-- // readonly, disabled --> <!-- display block : .block --> <span class="input-wrap block"> <input type="text" placeholder="이름을 입력하세요."> </span> <span class="input-wrap block"> <textarea placeholder="내용을 입력해 주세요."></textarea> </span> <!-- // display block : .block -->
