LG Update Center

설치 전 주의사항 및 사용방법 확인하기

   본 프로그램은

   Battery Miser Ver 4.0109.0420.01 로,

   XNOTE 에서 시스템의 상태를 체크하여 노트북 배터리 사용시간을

   최적으로 사용하기 위해 필요한 전원관리 프로그램입니다.

 

 

  [ 적용 모델 ]

 

  . XNOTE

  . Windows XP 사용 모델

 

 

  [ 주의 사항 ]

 

  ※ LG전자는 기본적으로 제품 구입시 적용된 OS 사용을 권장하며,

     이외의 OS 설치 후 발생하는 사용상의 문제에 대해서는

     책임지지 않습니다.

 

   - 적용 모델 이외의 모델에 설치시 오동작 할 수 있으므로,

     해당 모델을 확인하신 뒤 설치하세요.

 

   - 설치 방법은 사용환경에 따라서 약간 상이할 수 있습니다.

 

 

  [ 사용 방법 ]  

 

  1. 파일을 임시 폴더에 다운 받으세요.  

 

  2. 다운로드 받은 자료를 두번 클릭하여 압축을 해제합니다.

    임시폴더 내에 BatteryM 폴더가 생성됩니다.

 

  3. BatteryM\Setup.exe 를 실행하세요.

     

  4. BatteryMiser 설치 창이 나오면,

     화면의 지시대로 설치 진행합니다.
