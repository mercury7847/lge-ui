LG 모바일 스위치

나의 소중한 데이터 LG 모바일 스위치로 이어가세요
  • 모바일 스위치란?선택됨
  • 데이터 이동 가이드

주의하세요

데이터 이동은 스마트폰의 LG 모바일 스위치 앱으로 실행할 수 있습니다.
단, 일부 제품의 경우 LG 백업 앱으로 실행되며, 화면 구성에 차이가 있을 수 있습니다.

  • 타사 안드로이드 스마트폰 > LG 스마트폰선택됨
  • LG 스마트폰 > LG 스마트폰
  • IOS 스마트폰 > LG스마트폰
Step 1

LG 모바일 스위치 앱 설치 및 실행하기

기존 사용하시던 제품의 Play 스토어에 들어가셔서 LG 모바일 스위치 앱을 다운로드하여 설치하세요.
기존에 사용하시던 제품과 새 제품에서 LG 모바일 스위치 앱을 실행하세요.

Step 2

데이터 이동 방법 선택하기

기존 제품과 새 제품에서 LG 모바일 스위치 앱을 실행한 후 데이터를 이동할
방법을 선택하세요.

Step 3

USB 케이블/무선/ SD카드 중 선택하신 방법으로 데이터 이동하기

USB 케이블로 데이터 이동하기
  1. 1. 기존 제품에서 LG 모바일 스위치 앱을 실행한 후 USB 케이블을 누르세요.
  2. 2. 기존 제품과 새 제품을 USB 케이블로 연결하세요.
@@include('./common/modules/bullet-list.html', { "bulletTxt" : [ "제품의 USB 연결 방식을 파일 전송 (또는 MTP)로 선택하세요.", "제품에 따라 포트 타입이 다를 수 있으므로 각 제품에 적합한 OTG 젠더를 사용하세요." ] })
  1. 3. 연결이 완료되면 USB 케이블 받기 안내 창이 양쪽 제품 화면에 나타납니다. * 일부 제품의 경우 USB 케이블 연결 후에 기존 제품에서 접근 허용 또는 신뢰를 눌러주세요.
  2. 4. 기존 제품에서 보낼 데이터를 선택한 후 다음 버튼을 누르세요.
  1. 5. 받기가 완료되면 새 제품에 휴대폰 다시 시작을 눌러 새 제품을 다시 시작하세요. *정책에 따라 OTG 젠더는 기본 구성품으로 제공되지 않을 수 있습니다.

무선으로 데이터 이동하기

  1. 1. 새 제품에서 LG 모바일 스위치 앱을 실행한 후 무선 받기 시작을 누르세요.
  2. 2. 기존 제품에서 LG 모바일 스위치 앱을 실행한 후 무선 시작을 누르세요.
  3. 3. 기존 제품에서 데이터를 받을 새 제품의 기기 명칭을 선택하세요.
  1. 4. 새 제품에서 받기를 선택하세요.
  2. 5. 기존 제품에서 보낼 데이터를 선택한 후 다음 버튼을 누르세요.
  3. 6. 받기가 완료되면 새 제품에서 휴대폰 다시 시작을 눌러 새 제품을 다시 시작하세요.

SD 카드로 데이터 이동하기

  1. 1. 기존 제품에서 LG 모바일 스위치 앱을 실행하세요.
  2. 2. SD 카드를 눌러 보낼 데이터를 선택하고 시작을 눌러 데이터를 이동한 후, 저장이 완료되면 완료 버튼을 누르세요. * 기존 제품에 장착된 SD 카드에 데이터가 저장됩니다.
