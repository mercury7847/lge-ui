설치 전 주의사항 및 사용방법 확인하기

   본 프로그램은
   Battery Miser Ver 3.14 로, XNOTE 에서 시스템의 상태를 체크하여
   노트북 배터리 사용시간을 최적으로 사용하기 위해 필요한 프로그램입니다.

 

   [ 적용 모델 ]

 

   . LM40a / LM50a 모델
   . Windows 2000 / XP 사용 모델


   [ 주의 사항 ]
 
   - 적용 모델 이외의 모델에 설치시 오동작 할 수 있으므로,
     해당 모델을 확인하신 뒤 설치하세요.

 

   - 설치 방법은 사용환경에 따라서 약간 상이할 수 있습니다.

 


   [ 사용 방법 ]

 

   1. 파일을 임시 폴더에 다운 받으세요.

 

   2. 다운로드 받은 자료를 두번 클릭하여 압축을 해제합니다.
      임시폴더 내에 BatteryM 폴더가 생성됩니다.

 

   3. BatteryM\Setup.exe를 실행하세요.

    → 이미 프로그램이 설치된 경우엔,
      '프로그램 제거 확인'창이 뜨고, [확인]을 누르면 프로그램이 제거됩니다.
      '제거종료'화면이 나오면, [Finish]를 누르세요.
   
     다시, BatteryM\Setup.exe를 실행하세요.

 

   4. Battery miser 3.14 설치 화면이 나오면, [Next]를 누르세요.

 

   5. 나타나는 화면의 지시에 따라 계속 [Next]를 눌러 설치를 진행합니다.

 

   6. 프로그램 복사가 진행되고, '설치종료' 화면이 나오면, [Finish]를 누르세요.

 

   ※ 작업표시줄 우측 하단에 'Battery miser' 아이콘이 추가되었는지 확인합니다.
