01 쉽고 빠르게 데이터 이동하기

이전에 사용 중이던 제품 속 주소록, 메시지, 사진, 동영상 등의 데이터를 쉽고 빠르게 새 제품으로 옮길 수 있습니다.

아래 해당하는 경우를 선택해주세요.

쉽고 빠르게 데이터를 이동할 수 있는 가이드를 제공해드립니다.

  • 타사 안드로이드 스마트폰에서 LG 스마트폰으로 이동하기
  • LG 스마트폰에서 LG 스마트폰으로 이동하기
  • iOS 스마트폰에서 LG 스마트폰으로 이동하기

타사 안드로이드 스마트폰에서 LG 스마트폰으로 데이터 이동하는 방법

Step1 : LG 모바일 스위치 앱 설치 및 실행하기
기존 사용하시던 제품의 Play 스토어에 들어가셔서 LG 모바일 스위치 앱을 다운로드하여 설치하세요. 기존에 사용하시던 제품과 새 제품에서 LG 모바일 스위치 앱을 실행하세요.
Step1 : 데이터 이동하는 방법 선택하기
기존 제품과 새 제품에서 LG 모바일 스위치 앱을 실행한 후 데이터를 이동할 방법을 선택하세요.
Step3 : USB 케이블/무선/ SD카드 중 선택하신 방법으로 데이터 이동하기

USB 케이블로 데이터 이동하기

  1. 1. 기존 제품에서 LG 모바일 스위치 앱을 실행한 후 USB 케이블을 누르세요.
  2. 2. 기존 제품과 새 제품을 USB 케이블로 연결하세요.
    • *제품의 USB 연결 방식을 파일 전송 (또는 MTP)로 선택하세요.
    • *제품에 따라 포트 타입이 다를 수 있으므로 각 제품에 적합한 OTG 젠더를 사용하세요.
  3. 3. 연결이 완료되면 USB 케이블 받기 안내 창이 양쪽 제품 화면에 나타납니다.

    * 일부 제품의 경우 USB 케이블 연결 후에 기존 제품에서 접근 허용 또는 신뢰를 눌러주세요.

  4. 4. 기존 제품에서 보낼 데이터를 선택한 후 다음 버튼을 누르세요.
  5. 5. 받기가 완료되면 새 제품에서 휴대폰 다시 시작을 눌러 새 제품을 다시 시작하세요.

    *정책에 따라 OTG 젠더는 기본 구성품으로 제공되지 않을 수 있습니다.

무선으로 데이터 이동하기

  1. 1. 새 제품에서 LG 모바일 스위치 앱을 실행한 후 무선 받기 시작을 누르세요.
  2. 2. 기존 제품에서 LG 모바일 스위치 앱을 실행한 후 무선 시작을 누르세요.
  3. 3. 기존 제품에서 데이터를 받을 새 제품의 기기 명칭을 선택하세요.
  4. 4. 새 제품에서 받기를 선택하세요.
  5. 5. 기존 제품에서 보낼 데이터를 선택한 후 다음 버튼을 누르세요.
  6. 6. 받기가 완료되면 새 제품에서 휴대폰 다시 시작을 눌러 새 제품을 다시 시작하세요.

    * 무선으로 데이터를 이동하는 경우 기존 제품과 새 제품을 가까이에 위치시키세요.

SD 카드로 데이터 이동하기

  1. 1. 기존 제품에서 LG 모바일 스위치 앱을 실행하세요.
  2. 2. SD 카드를 눌러 보낼 데이터를 선택하고 시작을 눌러 데이터를 이동한 후, 저장이 완료되면 완료 버튼을 누르세요.

    * 기존 제품에 장착된 SD 카드에 데이터가 저장됩니다.

  3. 3. 기존 제품에서 마운트 해제 버튼을 누른 후 SD 카드를 빼내어 새 제품에 끼우세요.
  4. 4. 새 제품에서 LG 모바일 스위치 앱을 실행한 후 SD 카드를 누르세요.
  5. 5. 복원을 눌러 새 제품에 복원할 데이터를 선택한 후 다음 복원을 누르세요. 코멘트 : 해당 이미지
  6. 6. 복원이 완료되면 새 제품에서 휴대폰 다시 시작을 눌러 새 제품을 다시 시작하세요.

    * 사용하는 제품에 따라 SD 카드의 마운트 해제 방법 또는 경로가 다를 수 있습니다.

LG 스마트폰에서 LG 스마트폰으로 데이터 이동하는 방법

Step1 : LG 모바일 스위치 앱 업데이트 및 실행하기
기존 사용하시던 제품의 설정에 들어가셔서 일반 > 휴대폰 정보 > 앱 업데이트 로 진입하셔서 LG 모바일 스위치 메뉴가 최신인지 확인하시고 업데이트 안내가 있을 경우 업데이트 하셔서 최신 버전으로 업그레이드 해주세요. 기존에 사용하던 제품과 새 제품에서 LG 모바일 스위치 앱을 실행하세요.
Step2 : 데이터 이동하는 방법 선택하기
기존 제품과 새 제품에서 LG 모바일 스위치 앱을 실행한 후 데이터를 이동할 방법을 선택하세요.

02 이동 가능한 데이터 종류

주소록, 메시지, 통화기록, 캘린더, 음성녹음, 사진, 동영상, 음악, 문서, 메모, 다운로드앱,공인인증서, 기타 설정 항목 등을 새로운 LG 스마트폰으로 옮길 수 있습니다.

  • * LG전자 제품 간, 또는 LG전자 제품과 타사 제품 간의 이동 가능한 데이터 종류는 제조사, SW 버전, OS 종류 및 앱이나 콘텐츠 특성에 따라 제한되거나 달라질 수 있습니다.
  • * 각 제품의 SW 버전이나 앱 업데이트, OS 업그레이드, 제조사의 정책 등에 따라 본 설명서의 내용과 그림은 일부 달라질 수 있습니다.
