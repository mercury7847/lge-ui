- 수정 끝 주석 : <!-- //yymmdd_수정_xx -->

6. 이미지 관련

공통사항

  1. 파일명은 기본 SEO기준에 맞게 작성하되, 작은 해상도에 적용되는 이미지는 이미지 파일명 끝에 -m을 적용하여 구분한다.
  2. 이미지 기본 파일명은 디자인 파일 기준으로 기획에서 이미지 파일명 정리된 문서를 통해서 전달 후 퍼블리싱 작업 시 적용한다.
  3. 작은 해상도에 적용되어야 할 이미지는 디자인 파트에서 판단하여 2개의 이미지로 갈 경우 디자인 파트에서 전달합니다.
  4. 2개의 이미지가 적용되어야 할 경우 아래의 형태로 html소스에 적용하며, ui_responsive_image클래스는 js전용 클래스이므로 디자인 관련된 css작성 시 사용하지 않습니다
